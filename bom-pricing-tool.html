App Info
How this works
Web search: Browser fetches distributor pages, LLM analyzes them to find and verify product data.
Mouser API: Direct API call — returns structured JSON, no LLM needed. Get a free key at mouser.com/api-hub.
DuckDuckGo: Browser searches DDG for the part, identifies product pages, LLM visits each via tool-calling and extracts pricing from up to 15 sources.

Prerequisites
• Chrome with Allow CORS extension enabled (set Origin mode to ORIGIN)
• Visit the distributor site once (establishes cookies) — for Web search method
• LM Studio running at 127.0.0.1:1234 with a model loaded (8B+ recommended)
• In LM Studio: Developer tab → Server settings → Enable CORS
• For DuckDuckGo search: model must support tool/function calling (e.g. Qwen 2.5+, Llama 3.1+). Enable the visit-website tool in LM Studio.
• Mouser API: no LLM needed, just a free API key

Strict matching
The MPN on the product page must exactly match what you searched — character for character. Manufacturer must match if specified (abbreviations OK: TI, ST, ON, etc.). "Not found" is valuable — it flags BOM errors or discontinued parts.

Pricing logic
Build qty × line qty determines which price tier is used. Price Extended = Price Each × line qty (not × build qty). For example: qty 1, build qty 1000 → use the 1000-unit price tier, but extended = price × 1.

Fractional quantity (min buy qty > 1)
When quantity contains a decimal point (e.g. 1.0, 0.001), it indicates the portion of a minimum-buy package used per unit. For example: qty 0.001 with min buy 2500 means each unit uses 0.001 × 2500 = 2.5 components from a box of 2500. The tool calculates the number of boxes needed for the full build, spreads the total cost across all units, and shows the adjusted price per unit. A comment is added showing the box count and total min qty cost. Note: To trigger fractional handling, format the qty cell with decimal places in Excel (e.g. "0.00") — a value of "1" (no decimal) is treated as 1 actual component, while "1.00" (with decimals) is treated as 1 full package.
