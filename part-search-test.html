How this works
Web search: Browser fetches distributor pages, LLM analyzes them to find and verify product data.
Mouser API: Direct API call — returns structured JSON, no LLM needed. Get a free key at mouser.com/api-hub.

Prerequisites
• Chrome with Allow CORS extension enabled
• Visit the distributor site once (establishes cookies) — for Web search method
• LM Studio running at 127.0.0.1:1234 with a model loaded (8B+ recommended)
• In LM Studio: Developer tab → Server settings → Enable CORS (required for browser requests)
• For DuckDuckGo search: LM Studio model must support tool/function calling (e.g. Qwen 2.5+, Llama 3.1+). Enable the visit-website tool in LM Studio. CORS extension required for page fetching.

Strict matching
The MPN on the product page must exactly match what you searched — character for character. Manufacturer must match if specified (abbreviations OK). "Not found" is valuable — it flags BOM errors or discontinued parts.
